Get started with Bootstrap

Introduction

Bootstrap is a popular and widely-used open-source front-end framework for web development. It was originally created by Twitter and is now maintained by the Bootstrap community. This framework simplifies the process of designing and building responsive and attractive web pages and web applications. Bootstrap is known for its pre-designed, customizable components and responsive grid system, which makes it easier for developers to create visually appealing and functional websites.

Key features of Bootstrap include:

  1. Responsive Design: Bootstrap's grid system allows developers to create responsive web layouts that adapt to different screen sizes, ensuring a consistent user experience on various devices, from desktops to smartphones.
  2. Pre-Designed Components: Bootstrap provides a rich set of pre-designed components, such as buttons, navigation bars, forms, modals, and more. These components are easily customizable to match your project's style.
  3. CSS and JavaScript Framework: Bootstrap includes a robust set of CSS and JavaScript files that help streamline the development process. It also offers utility classes for common tasks like spacing and typography.
  4. Browser Compatibility: Bootstrap is designed to work well with modern web browsers and maintains backward compatibility with older versions, ensuring a wide reach for your web projects.
  5. xtensibility: While Bootstrap offers a comprehensive set of components and styles, you can extend it further by creating custom themes or integrating third-party plugins.
  6. Active Community: The Bootstrap community is active and engaged, providing support, documentation, and numerous resources for developers.

Breakpoints

Breakpoints are customizable widths that determine how your responsive layout behaves across device or viewport sizes in Bootstrap.

  • Extra Small (xs): This is the smallest breakpoint and targets devices with very narrow screens, such as smartphones. The CSS class prefix for this breakpoint is col-xs-.
  • Small (sm): The small breakpoint is designed for devices like small tablets and larger smartphones. The CSS class prefix for this breakpoint is col-sm-.
  • Medium (md): This breakpoint is suitable for medium-sized devices, such as larger tablets and small laptops. The CSS class prefix for this breakpoint is col-md-.
  • Large (lg): The large breakpoint is intended for desktop screens and larger laptops. The CSS class prefix for this breakpoint is col-lg-.
  • Extra Large (xl): The extra-large breakpoint targets larger desktop displays. The CSS class prefix for this breakpoint is col-xl-.

HTML

<div class="container">
<div class="row">
<div class="col-xs-12 col-sm-6 col-md-4 col-lg-3 col-xl-2">
<!-- Content for different screen sizes --> </div>
<div class="col-xs-12 col-sm-6 col-md-4 col-lg-3 col-xl-2">
<!-- Content for different screen sizes --> </div>
<!-- ... more columns ... --> </div>
</div>

Grid

The grid system in Bootstrap is a powerful and flexible layout system that allows web developers to create responsive and organized web page layouts. Bootstrap's grid system is based on a 12-column layout and uses CSS classes to define how content is displayed across different screen sizes, making it a key component of creating responsive designs.

Structure

The grid system consists of 12 columns by default. You can customize the number of columns in Bootstrap if needed. Rows are used to group and align content within a container. Each row can have a maximum of 12 columns. Columns are the building blocks of the grid system, and they should always be placed inside a row.

Column Classes

Bootstrap provides classes like col-, col-sm-, col-md-, col-lg-, and col-xl- to define the number of columns a content block should span at different screen sizes (breakpoints). For example, col-6 will make an element span half the width of its parent container on all screen sizes. col-sm-4 may make the same element occupy one-third of the width on small screens.

Form

>Forms are a crucial part of web development, and Bootstrap offers a comprehensive set of classes and components to streamline the process of creating attractive and functional forms.

  • Form Structure:
  • Bootstrap forms are built using HTML form elements. You can create basic forms, inline forms, and horizontal forms with varying structures. The form element is often wrapped in a div with the class .container, .container-fluid, or other grid-related classes to control form layout.
  • Form Controls:
  • Bootstrap provides classes for various form controls, including text inputs, textareas, checkboxes, radio buttons, select dropdowns, and file inputs. These controls can be customized using classes like .form-control to ensure consistency in appearance and functionality.
  • Form Utilities:
  • Bootstrap provides various utility classes for styling form elements, such as .disabled, .readonly, and .plaintext. These classes help customize the appearance and behavior of form controls.

Form-Control

In Bootstrap, form controls are a key part of creating well-styled and functional forms on your web pages. Form controls are HTML elements like text inputs, checkboxes, radio buttons, select dropdowns, and more, which Bootstrap enhances with its own CSS classes and JavaScript for added functionality and consistency. Here's an overview of form controls in Bootstrap:

1. Text Input

Bootstrap styles text inputs with the .form-control class. This class ensures that the text inputs have consistent sizing, padding, and border styling.

HTML

<input type="text" class="form-control" placeholder="Username">

2. Textarea:

Textareas in Bootstrap are styled similarly to text inputs. You can use the .form-control class for consistent styling.

HTML

<textarea class="form-control" placeholder="Enter your message"></textarea>

4. Checkboxes and Radio Buttons:

Bootstrap provides custom-styled checkboxes and radio buttons. You can use classes like .form-check and .form-check-input to create visually appealing options.

You can add the .disabled class to form controls to make them appear disabled. For readonly inputs, use the .readonly class. These classes help indicate the state of the control.

If you want to apply a custom shadow level, you can use the .shadow-{level} class, where {level} is the desired shadow level, such as 1, 2, 3, 4, or 5.

HTML

<div class="shadow-1">This div has a custom shadow level.</div>
<div class="shadow-2">This div has a custom shadow level.</div>
<div class="shadow-3">This div has a custom shadow level.</div>
<div class="shadow-4">This div has a custom shadow level.</div>

Spaces

Bootstrap includes a wide range of shorthand responsive margin, padding, and gap utility classes to modify an element’s appearance.

Margin & Padding

Where property is one of:

  • m - for classes that set margin
  • p - for classes that set padding

Where sides is one of:

  • t - for classes that set margin-top or padding-top
  • b - for classes that set margin-bottom or padding-bottom
  • s - (start) for classes that set margin-left or padding-left in LTR, margin-right or padding-right in RTL
  • e - (end) for classes that set margin-right or padding-right in LTR, margin-left or padding-left in RTL
  • x - for classes that set both *-left and *-right
  • y - for classes that set both *-top and *-bottom

Grid Gap

HTML

<div class="grid gap-0 row-gap-3">
<div class="p-2 g-col-6 bg-primary">Grid item 1</div>

Documentation and examples for common text utilities to control alignment, wrapping, weight, and more.

Text alignment

Easily realign text to components with text alignment classes. For start, end, and center alignment, responsive classes are available that use the same viewport width breakpoints as the grid system.

HTML

Start aligned text on all viewport sizes.

Center aligned text on all viewport sizes.

End aligned text on all viewport sizes.

<div class="grid gap-0 row-gap-3">
<p class="text-start">Start aligned text on all viewport sizes.</p>
<p class="text-center">Center aligned text on all viewport sizes.</p>
<p class="text-end">End aligned text on all viewport sizes.</p>
